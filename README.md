This project demonstrates Android UI development, localization, and interactive components based on the SimpleView1 module from Week 3.

## Project Overview
This app showcases a single module, LinhView3, with multiple interactive features and full support for English and French.

## Features
- Single module: `LinhView3`
- Minimum API 31 (Android 12), Target API 36/37
- All text stored in `strings.xml` (English & French)
- Custom AlertDialog on Android back button with icon, title, message, and Yes/No buttons
- Open device settings via button click
- ImageButton rotates through 4 images and shows a Toast with image name and student info
- EditText accepts alphabetic input only, max 10 characters, with hint
- Save Button launches a website
- Switch button shows Snackbar messages
- Logs full name and student ID when app goes to background
